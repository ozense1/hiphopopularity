# What Makes Hip-Hop Popular?

An interactive dashboard exploring what actually drives Spotify popularity for hip-hop tracks released between 1986 and 2023. It looks at audio features, lyric complexity, explicit content, subgenres, and Genius engagement.

Built with D3.js.

**Live site:** [https://ozense1.github.io/hiphopopularity/](https://ozense1.github.io/hiphopopularity/)
